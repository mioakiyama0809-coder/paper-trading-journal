# 📈 纸上交易日志

> A股波段交易学习工具 · 纯前端 · 本地存储 · 无需服务器

## 简介

本工具是一个纯前端的**纸上交易（模拟交易）记录与复盘系统**，专为 A 股波段交易学习者设计。所有数据存储在浏览器 `localStorage` 中，打开网页即可使用，无需注册、无需服务器。

**设计原则**：
- A 股配色规则：🔴 **红色 = 盈利/涨**，🟢 **绿色 = 亏损/跌**（与美股相反）
- 深色主题，护眼，适合盯盘后使用
- 响应式设计，手机和电脑均可使用

## 功能说明

### 1. 交易录入

- 填写日期、标的代码/名称、买入/卖出方向、价格、数量
- 记录**交易理由**（看到什么信号、什么逻辑）
- 设置**止损位**和**目标位**
- 可选填写备注

### 2. 交易记录列表

- 表格展示所有记录，按日期倒序排列
- 支持按标的搜索、按状态/方向筛选
- 已平仓记录自动显示盈亏金额和百分比（红=盈利，绿=亏损）
- 每条记录可**编辑**、**删除**、**平仓**、查看**详情**

### 3. 平仓结算

- 点击「平仓」按钮，填写卖出价格和日期
- 自动计算：
  - **盈亏金额** = (卖出价 - 买入价) × 数量
  - **盈亏百分比** = (卖出价 - 买入价) / 买入价 × 100%
  - **持仓天数**

### 4. 复盘总结

- 每条交易可填写复盘总结：判断对了还是错了？错在哪里？下次怎么做？
- 复盘内容保存在交易记录中

### 5. 统计仪表盘

- 总交易笔数、已平仓笔数、持仓中
- **胜率**（盈利笔数 / 已平仓总笔数）
- **盈亏比**（平均盈利 / 平均亏损绝对值）
- 总盈亏金额、平均盈利/亏损
- 最大单笔盈利/亏损
- 最长连续盈利/亏损次数

### 6. 图表分析

- 📊 **累计盈亏曲线**（折线图）
- 🥧 **胜率饼图**（盈利 vs 亏损笔数）
- 📦 **各标的总盈亏柱状图**

### 7. 数据管理

- **导出**：将数据导出为 JSON 文件备份
- **导入**：从 JSON 文件恢复/合并数据
- **清空**：清空所有数据（需二次确认）

## 推荐使用方式（3 个月练习计划）

| 阶段 | 时间 | 目标 |
|------|------|------|
| 观察 | 第 1-2 周 | 每天收盘后用「录入」记录模拟买卖，填写理由 |
| 积累 | 第 3 周起 | 持续记录，每次平仓后填写复盘总结 |
| 评估 | 第 3 个月末 | 查看统计：胜率 ≥ 55%？盈亏比 ≥ 1.5？考虑真实入场 |

> 💡 **最重要的是「交易理由」和「复盘总结」这两栏，没有这两栏的纸上交易没有意义。**

## 注意事项

- 数据存储在浏览器 `localStorage` 中，**清除浏览器数据会导致记录丢失**
- 请定期使用「数据管理 → 导出」功能备份数据
- 本工具为纯前端实现，所有数据仅存在于本地，不会上传到任何服务器
